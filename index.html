Jul. 2020

Automation Engineer Intern

OLI Sistemas Sanitários S.A. is a portuguese multinational and the largest cistern producer in Southern Europe, owning the OLI bath solutions brand, making it my first contact with a huge corporation.

This is where I developed my Master's thesis, whose main objective was to automatically update the counters of manufacturing orders of three assembly lines through an IoT solution that followed Industry 4.0 principles. This solution also dealt with validation of the current order.

A proof of concept of the solution was developed, with the stack starting with the existing PLC and Robot devices on the bottom. On top of it, I developed an algorithm on a custom edge device built by me (whose processing unit was an ESP32 microcontroller) that would interpret data from the existing PROFIBUS network, which would detect the end of the processing of a pallet and send this information to a RPi 3B+ via the company's network (using MQTT).

A MES like system was developed using Node-Red that would update these counts of the Company's Test Database, which had an exact copy of existing manufacturing orders. The MES would know the exact quantities of each pallet and what order to update through the validation of the orders at the beggining of the manufacturing processes, when a barcode would be scanned. Additionally, a Web UI and Dashboard was developed using Bootstrap Vue and Vue.Js to display production information: counters, which order was running, if validation failed, among others. It also had control capabilities, like changing the amount of units per pallet on the running orders (to account for an existing edge case).
